# Hospital-Health-care-performance-analysis-Dashboard
This project provides a comprehensive Overview of health care Performance by Analyzng key metrics and other patient and operational factors, to support data driven decision-making, optimise and improve overall health care service delivery

# Key Metrics
key metrics/KPI like treatment cost Total Patient Average length of stay were used in the analysis to solve question to make data-driven decisions

# Data Preparation
Transformed data using power query, Cleaned raw data, Handled inconsisternt values and created helper columns for analysis.

# Dashboard Features
KPI cards summarizing key performance indicators
interactive charts and visual
slicer for better data exploration
clean and user friendly layout
Readable fpr non-technical individual
sheets conected to the dashboard (brief, insights etc)

# How to use 
Download the Microsoft Excel file from this repository
Open it using Microsoft Excel
